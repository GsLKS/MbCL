# MbCL Downloader

* MbCL é um repositório de arquivos e bibliotecas online

### Baixando [MbCL](https://app.mbcl.ml):

* Via NPM (MacOSLinux/Windows):

```js
npm install -g mbcl
```

* Repositório Oficial (Linux):

clique [aqui](https://repo.mbcl.ml) para baixar via web

```
./mbcl --dl latest/mbcl
```

### Testando o pacote:

```
mbcl --version
```
